# Real de Miramar · Ruta de Valor

Tablero interno de tramitología del desarrollo Real de Miramar (Guaymas, Sonora),
co-desarrollo YoDesarrollo. Pide clave para verse y no se indexa.

Los datos se administran desde un Google Sheet vía Apps Script (`?recurso=board`);
nada sensible se hardcodea aquí. Los montos financieros y la escritura requieren
sus propias claves (no incluidas en este repo).
